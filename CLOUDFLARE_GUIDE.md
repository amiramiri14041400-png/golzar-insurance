# راهنمای پیاده‌سازی و استقرار وب‌سایت نمایندگی بیمه ایران روی Cloudflare

این پروژه شامل یک بخش فرانت‌اند React (طراحی شده با Vite و Tailwind CSS) و یک سرور بک‌اند Node.js/Express برای مدیریت استعلام‌ها، کد پیگیری و مشاور هوشمند (Gemini API) است.

برای پیاده‌سازی و انتشار وب‌سایت روی حساب **Cloudflare** خود، ۳ روش متناسب با نیاز شما وجود دارد:

---

## روش اول: استقرار فرانت‌اند روی Cloudflare Pages (پیشنهادی و سریع‌ترین)

اگر می‌خواهید فرانت‌اند پروژه به صورت مستقیم، رایگان و فوق‌العاده سریع روی دامنه‌تان در کلودفلر اجرا شود:

### ۱. خروجی کد پروژه
1. از منوی بالای صفحه **Export to GitHub** را بزنید یا فایل ZIP پروژه را دانلود کرده و در مخزن (Repository) گیت‌هاب خود آپلود کنید.

### ۲. اتصال به Cloudflare Pages
1. وارد پنل مدیریت **Cloudflare** خود شوید.
2. از منوی سمت چپ به بخش **Workers & Pages** بروید.
3. روی **Create Application** کلیک کرده و تب **Pages** را انتخاب کنید.
4. گزینه **Connect to Git** را بزنید و اکانت گیت‌هاب خود را متصل کنید.
5. ریپوزیتوری پروژه بیمه را انتخاب کنید.

### ۳. تنظیمات بیلد (Build Settings)
- **Framework Preset:** گزینه **`None`** (یا **`None / Static`** - زیرا Vite در لیست کشویی وجود ندارد یا فقط VitePress هست)
- **Build Command:** `npm run build`
- **Build Output Directory:** `dist` (اگر پروژه Vite/React است) یا `out` (اگر پروژه Next.js با تنظیم static export است)

### ۴. متغیرهای محیطی (Environment Variables)
در بخش **Environment Variables** در کلودفلر، موارد زیر را وارد کنید:
- `VITE_SUPABASE_URL`: (آدرس پروژه Supabase شما - در صورت استفاده)
- `VITE_SUPABASE_ANON_KEY`: (کلید عمومی Supabase - در صورت استفاده)

روی **Save and Deploy** کلیک کنید. پس از ۱ دقیقه وب‌سایت شما روی کلودفلر منتشر می‌شود!

---

## روش دوم: اجرای کامل (فرانت‌اند + بک‌اند Express) روی VPS با Cloudflare Tunnel

اگر می‌خواهید بخش بک‌اند پروژه (کدهای پیگیری، ذخیره استعلام‌ها و مشاور هوشمند Express) نیز فعال باشد:

### ۱. اجرای پروژه روی VPS یا سرور
روی سرور مجازی یا هاست Node.js خود دستورات زیر را اجرا کنید:
```bash
npm install
npm run build
npm start
```
پروژه روی پورت `3000` شروع به کار می‌کند.

### ۲. اتصال سرور به Cloudflare با Cloudflare Tunnel
1. در پنل کلودفلر به بخش **Zero Trust > Networks > Tunnels** بروید.
2. یک Tunnel جدید بسازید و دستور نصب `cloudflared` را روی سرور اجرا کنید.
3. در بخش Public Hostname، دامنه خود (مثلاً `bimeh30026.ir`) را به پورت محلی سرور متصل کنید:
   - **Service:** `HTTP`
   - **URL:** `localhost:3000`

با این روش، کلودفلر به عنوان لایه امنیتی و CDN جلوتر از سرور شما قرار می‌گیرد و SSL خودکار فعال می‌شود.

---

## روش سوم: استفاده از Supabase برای دیتابیس + Cloudflare Pages برای فرانت‌اند

اگر نمی‌خواهید سرور VPS نگهداری کنید:
1. دیتابیس پروژه را به **Supabase** متصل کنید.
2. کل پروژه فرانت‌اند را طبق **روش اول** روی **Cloudflare Pages** آپلود کنید.
3. کدهای دریافت استعلام و ثبت نام مستقیم از طریق جاوااسکریپت با Supabase ارتباط برقرار خواهند کرد.

---

### پشتیبانی و تنظیمات دامنه اختصاصی
پس از استقرار در Cloudflare Pages، از تب **Custom Domains** می‌توانید دامنه شخصی خود (مثلاً `bimeh-golzar.ir`) را با یک کلیک ست کنید.
